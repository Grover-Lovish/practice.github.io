Web Technologies

World Wide Web

WWW stands for world wide web.
The technical definition of world wide web is : all the resources and users on the internet that are using the Hypertext transfer Protocol (HTTP)
In simmple terms The World Wide Web is the way of exchanging information between computers on the internet, typing them together into a vast collection of interactive multimedia resources.
Internet

Internet is a global information network that collects millions of computers. It is growing exponentially and provides a unique information resource that is global, diverse and current.
Evolution

World Wide Web was created by Timothy Berners Lee in 1989 at CERN in Geneva. World Wide Web came into existence as a proposal by him to allow researchers to work together effectively and efficiently at CERN, Eventually it becomes World Wide Web.

Web Page

Web Page is a document available on World Wide Web. Web Pages are stored on web server and can be view using web browser.
A web page can contain huge information including text, graphics, audio, video and hyper links. These hyper links are the link to other web pages
Website

A website us a collection of web pages and related content that is identified by a common domain name and published on at least one web server. Notable example are wikipedia.org, google.org and amazon.org
Websites are typically dedicated to a particular topic or purpose, such as, news, education, commerce, entertainment, or social networking. Hyperlinking between webpages guides the navigation of the site, which often starts with a home page.
Users can access websites on a range of devices, including desktops, laptops, tablets and smartphones. The app used on these devices is called web browser
Static and Dynamic Website

Static Website


  • Prebuilt Content is same every time the page is loaded.
  • It use the HTML code for developing a website
  • It sends exactly the same response for every request.
  • The content is only changed when someone publishes and updates the file (send it to the web server).
  • Flexibility is the main advantage of static websites.

Dynamic Website


  • Content is generated quickly and changes regularly.
  • It uses the server side languages for developing a website
  • It may generate different HTML for each of the request.
  • The page contains "server side" code which allows the server to generate the unique content when the page is loaded.
  • Content Management System (CMS) is the main advantage of dynamic website.
Webpage vs Homepage

A Homepage is a specific type of a webpage for facilitating user engagement.For example a homepage can highlight the most recent articles of a blog or online journal, provide visitors with the most relevant reading material upfront.It can also be a virtual place for promting events and special offers and updating visitors.
Whereas the term website refers to any page of collected content displayed on a website. A website will typically include multiple webpages linked together in an orderly way, and these are often accessed through the homepage.
